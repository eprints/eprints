TITLE: About the Archive

This archive is running on eprints.org open archive software, a freely distributable archive system available from eprints.org. The open archives interoperability is achieved using open archives software developed at Cornell university.

Other institutions are invited (and encouraged) to set up their own open archives for author self-archiving, using the freely-distributable eprints.org software used at this site.
